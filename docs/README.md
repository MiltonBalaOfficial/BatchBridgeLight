# Project Documentation

This document provides a high-level overview of the `BatchBridge` project, a platform for MBBS students of College of Medicine and JNM Hospital (COMJNMH) to connect with their seniors, juniors, and batchmates, fostering inter-batch bonding and sharing experiences. It links to more detailed documentation on specific areas.

## Project Overview

This is a Next.js project utilizing the App Router, written in TypeScript. It integrates Tailwind CSS for styling, Shadcn UI for components, and Clerk for authentication.
